# Тестовое задание каталог интернет-магазина (Laravel 11)

Тестовое задание: каталог товаров с группами, сортировкой и пагинацией на основе БД (`groups`, `products`, `prices`).

---

## Функционал

### Главная страница
- Список корневых групп (где `id_parent = 0`)
- У каждой группы отображается **общее количество товаров** в этой группе **и всех вложенных подгруппах**
- Список товаров (все товары или в рамках выбранной группы) с:
  - сортировкой по цене (ASC/DESC)
  - сортировкой по названию (ASC/DESC)
  - пагинацией

### Страница группы
- Показывает подгруппы выбранной группы
- Показывает товары из выбранной группы **и всех её подгрупп**
- Сортировка и пагинация работают **только в рамках выбранного дерева**

### Карточка товара
- Название товара
- Цена (используется минимальная цена `MIN(price)` если цен несколько)
- Хлебные крошки (вся цепочка родительских групп)

### UI
- Bootstrap
- Сетка товаров 
- AJAX-пагинация
---

## Тестирование

Тесты полностью изолированы друг от друга.
Для каждого теста используется чистая база данных (SQLite in-memory),
которая пересоздаётся благодаря трейту RefreshDatabase.

Это гарантирует:
- отсутствие побочных эффектов
- независимость тестов
- стабильность выполнения

## Быстрый старт (Docker)

### 1) Поднять контейнеры
```bash
docker compose up -d --build
```
### 2) Установить зависимости (если нужно)
```bash
docker compose exec app composer install
```
### 3) Настроить окружение
```bash
docker compose exec app cp .env.example .env
docker compose exec app php artisan key:generate
```
### 4) Выполнить миграции
```bash
docker compose exec app php artisan migrate
```
### 5)Заполнить базу данными боевыми/демо данными
```bash
docker compose exec app php artisan db:seed --class=SqlDumpSeeder
```
### 6) Тесты 
```bash
docker compose exec app php artisan test
```
